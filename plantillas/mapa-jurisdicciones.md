# Mapa de jurisdicciones

> Quién manda en qué. Evita que dos agentes se pisen (invariante 5). Es la encarnación ligera del reparto que también viven los `role.md`; para quien usa forge, `CODEOWNERS` es su equivalente ejecutable.

| Jurisdicción (zona / carpeta / función) | Agente responsable | Puede tocar | No puede tocar | Cómo pedir cruzar |
|---|---|---|---|---|
| *Ejemplo:* `/docs` y textos del método | *(coordinadora)* | documentos | código de producto | lo pide a la dirección |
| *Ejemplo:* `/web` | *(agente front)* | la web | el método y los secretos | `Handoff a:` + la coordinadora arbitra |
|  |  |  |  |  |

**Regla:** salir de tu jurisdicción se pide antes (`Handoff a: <agente>` en bitácora); no se entra por cuenta propia. La coordinadora arbitra; la dirección humana tiene la última palabra. Redibujar este mapa es diseño de equipo: lo ratifica la humana (§VIII-B de la carta).
